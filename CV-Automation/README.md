# CV Automation Workflow for AI-Based Candidate Screening

AI-powered recruitment automation workflow built using n8n for intelligent candidate screening and CV analysis.

## Overview

This workflow automates the process of collecting candidate CVs, extracting resume content, analyzing qualifications using AI, and storing candidate evaluation results automatically.

## Features

- Automated CV submission handling
- Resume text extraction
- AI-based qualification analysis
- Candidate scoring system
- Google Sheets integration
- Recruitment workflow automation

## Workflow Architecture

1. Application Form Submission
2. CV Upload Processing
3. Resume Text Extraction
4. AI Qualification Analysis
5. Candidate Scoring
6. Data Logging in Google Sheets

## Technologies Used

- n8n
- Groq AI
- Google Gemini AI
- Google Sheets API
- OCR/Text Extraction
- AI Prompt Engineering
- Workflow Automation

## Use Cases

- Automated recruitment screening
- AI-based HR automation
- Resume qualification analysis
- Candidate shortlisting
- Smart hiring workflows

## Project Outcome

Successfully developed an AI-powered recruitment workflow capable of automatically analyzing candidate resumes and generating qualification insights with minimal manual effort.
