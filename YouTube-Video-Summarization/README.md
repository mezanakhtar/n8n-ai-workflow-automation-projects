# YouTube Video Summarization Workflow

AI-powered workflow for extracting and summarizing YouTube video transcripts using n8n automation.

## Overview

This workflow automatically retrieves YouTube video transcripts, processes the text using AI models, and generates concise summaries for quick content understanding.

## Features

- YouTube transcript extraction
- AI-generated summaries
- Automated content processing
- Long video summarization
- AI text analysis
- Workflow automation

## Workflow Architecture

1. YouTube URL Input
2. Transcript Retrieval
3. Transcript Processing
4. Text Chunking
5. AI Summarization
6. Summary Output

## Technologies Used

- n8n
- Groq AI
- Google Gemini AI
- YouTube Transcript API
- AI Summarization
- Workflow Automation

## Use Cases

- Educational video summaries
- Research automation
- Content analysis
- Quick learning workflows
- AI-powered media summarization

## Project Outcome

Developed an efficient AI workflow capable of transforming lengthy YouTube videos into concise and meaningful summaries automatically.
