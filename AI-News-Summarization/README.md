# AI News Summarization Workflow

AI-powered news aggregation and summarization workflow built using n8n automation.

## Overview

This workflow automatically collects the latest news from multiple RSS feeds, processes the content using AI models, and generates concise summaries that are delivered through Gmail.

## Features

- Automated RSS news collection
- AI-generated news summaries
- Tech and world news categorization
- Gmail delivery automation
- Smart content filtering
- Automated news aggregation

## Workflow Architecture

1. Workflow Trigger
2. RSS Feed Collection
3. News Aggregation
4. AI Content Analysis
5. AI Summarization
6. Gmail Summary Delivery

## Technologies Used

- n8n
- Google Gemini AI
- RSS Feed Integration
- Gmail API
- AI Prompt Engineering
- Workflow Automation

## Use Cases

- Daily news digest automation
- AI-powered news analysis
- Automated content summarization
- Personalized news delivery
- Information aggregation systems

## Project Outcome

Built an automated AI news assistant capable of collecting, analyzing, and summarizing news content efficiently using workflow automation and AI technologies.
