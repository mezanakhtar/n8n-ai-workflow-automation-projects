# n8n AI Workflow Automation Projects

This repository contains AI-powered workflow automation projects developed using n8n during my internship training. These projects demonstrate practical implementations of Generative AI, workflow orchestration, automation pipelines, and API integrations using modern AI tools and services.

# Projects Included

## 1. CV Automation Workflow

An AI-powered candidate screening and CV analysis workflow.

### Features
- Extracts text from uploaded CVs
- Uses Groq LLM for candidate evaluation
- Performs AI-based qualification analysis
- Stores candidate results in Google Sheets
- Automates recruitment screening process

### Technologies Used
- n8n
- Groq LLM
- Google Sheets API
- AI Prompt Engineering
- Workflow Automation

## 2. AI News Summarization Workflow

An automated AI news aggregation and summarization system.

### Features
- Collects latest AI and technology news from RSS feeds
- Uses Gemini AI to summarize news articles
- Categorizes important updates
- Sends summarized reports automatically through Gmail

### Technologies Used
- n8n
- Google Gemini AI

- Gmail API
- AI Summarization

- Enables semantic document search and contextual responses
