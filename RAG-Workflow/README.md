# RAG Workflow for Company Documents

AI-powered Retrieval-Augmented Generation (RAG) workflow built using n8n for intelligent document retrieval and question answering.

## Overview

This workflow processes company documents stored in Google Drive, converts them into embeddings, stores them in a vector database, and enables AI-powered semantic search and question answering.

## Features

- Google Drive document integration
- AI-powered document retrieval
- Semantic vector search
- Question answering system
- Automated document processing
- Context-aware AI responses

## Workflow Architecture

1. Google Drive Trigger
2. Document Extraction
3. Text Chunking
4. Embedding Generation
5. Vector Database Storage
6. AI Retrieval
7. AI Response Generation

## Technologies Used

- n8n
- Google Drive API
- Pinecone Vector Database
- Google Gemini AI
- RAG Architecture
- Embedding Models
- AI Automation

## Use Cases

- Company knowledge base
- Internal documentation assistant
- AI-powered search engine
- Smart document retrieval
- Enterprise AI assistant

## Project Outcome

Successfully implemented a scalable AI retrieval system capable of understanding and answering questions from company documents using semantic search and AI reasoning.
